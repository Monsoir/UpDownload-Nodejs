# 简单的上传下载服务器 Node.js

开发环境：Node.js v8.5.0

运行方法：

1. 进入到 package.json 所在的文件夹
2. 安装依赖

    ```sh
    npm install
    ```

3. 运行服务器

    ```sh
    npm start
    ```

## API

API | 说明
--- | ---
127.0.0.1:3000/api/files | 检索文件
127.0.0.1:3000/api/upload | 上传
127.0.0.1:3000/api/download/<file name> | 下载
